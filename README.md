# ZYNCO Africa

A modern corporate website for ZYNCO Systems Ltd - Technology and business solutions for enterprises, institutions, and growing businesses in East Africa.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
